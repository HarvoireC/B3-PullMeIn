À propos de PullMeIn

PullMeIn est une plateforme communautaire dédiée au partage d’articles techniques rédigés par des professionnels et des passionnés. Le projet sert également de support pédagogique pour illustrer un vrai workflow d’intégration continue.

Objectifs du projet

  • Proposer une expérience de contribution simple via un formulaire en ligne.
  • Automatiser la création de Pull Requests et la validation des contenus avec GitHub Actions.
  • Assurer la qualité éditoriale et la sécurité (vérifications automatiques du contenu).
  • Déployer un site statique moderne via GitHub Pages.

Comment ça fonctionne ?

  1. L’utilisateur remplit le formulaire sur la page “Proposer un article”.
  2. Une API Node.js hébergée sur Render reçoit les données et crée automatiquement une Pull Request sur GitHub.
  3. Une pipeline GitHub Actions vérifie les fichiers JSON (champs obligatoires, liens, sécurité, etc.) et teste la génération des pages HTML.
  4. Une fois la Pull Request validée et mergée, une autre pipeline génère les pages d’articles et met à jour la liste des articles.
  5. Le site est ensuite servi par GitHub Pages, avec le nouvel article disponible en ligne.

Ce que le prototype démontre

  • Un site statique HTML/CSS/JavaScript déployé sur GitHub Pages.
  • Une API externe (Render) qui crée des Pull Requests à partir d’un simple formulaire.
  • Des pipelines CI/CD pour valider et générer le contenu (articles HTML, liste des articles).
  • Une gestion simple de l’utilisateur côté front (mode invité, pseudo en localStorage).
